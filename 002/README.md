# Exercise: Create a program that shows your name, address and also has some comments in it

- Use the fmt package

```golang
package main

import "fmt"

func main () {
	// Here goes your code
}
```

<details>
<summary> Solution: </summary>

```golang
package main

import "fmt"

func main () {
    // Printing my name
    fmt.Println("My Name is Enin")
    // Printing my address!
    fmt.Println("My address is: Summs Rift 42")
}

// To run the program:
// - go run solution.go
```
